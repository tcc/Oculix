## What is OculiX?

**OculiX is a visual automation IDE** that drives any GUI by what it looks like — not by accessibility hooks, DOM selectors, or fragile XPaths. It sees the screen the way a human does, then clicks, types, and reads exactly where it should.

Forked from the long-standing SikuliX project and re-engineered around an embedded OCR stack, a modern vision pipeline, and a clean Jython scripting layer, OculiX is what visual testing should have been in 2026.

<CardGrid stagger>
  <Card title="Pixel-perfect matching" icon="seti:image">
    Find any region of any window by sample image. Resolution-aware, similarity-tunable, sub-pixel reliable.
  </Card>
  <Card title="OCR out of the box" icon="document">
    Tesseract is **embedded** via Legerix — no manual install. PaddleOCR available as an opt-in HTTP server for high-accuracy CJK and complex layouts.
  </Card>
  <Card title="Cross-platform" icon="laptop">
    Windows, macOS, Linux. One script runs everywhere — no platform-specific selectors.
  </Card>
  <Card title="Jython scripting" icon="seti:python">
    Python 2.7 syntax with full JVM interop. Drop-in compatible with legacy SikuliX scripts.
  </Card>
  <Card title="Vision pipeline" icon="puzzle">
    Modern computer-vision primitives (template, feature, contour) composable from the IDE or from code.
  </Card>
  <Card title="Vendor-of-record support" icon="approve-check">
    Open source for everyone. Commercial support and compliance contracts available for regulated industries.
  </Card>
</CardGrid>
