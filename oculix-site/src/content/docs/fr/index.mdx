## Qu'est-ce qu'OculiX ?

**OculiX est un IDE d'automatisation visuelle** qui pilote n'importe quelle interface graphique par ce qu'elle affiche — sans hooks d'accessibilité, sans sélecteurs DOM, sans XPath fragiles. Il voit l'écran comme un humain, puis clique, tape et lit exactement au bon endroit.

Fork du projet historique SikuliX, ré-architecturé autour d'un OCR embarqué, d'un pipeline de vision moderne et d'une couche de scripting Jython propre, OculiX est ce que les tests visuels auraient dû être en 2026.

<CardGrid stagger>
  <Card title="Reconnaissance pixel-perfect" icon="seti:image">
    Repère n'importe quelle zone d'une fenêtre à partir d'une image-échantillon. Adaptatif à la résolution, similarité ajustable, fiable au sous-pixel près.

  <Card title="OCR clé en main" icon="document">

  <Card title="Multi-plateforme" icon="laptop">
    Windows, macOS, Linux. Un script unique tourne partout — aucun sélecteur spécifique à une plateforme.
  </Card>
  <Card title="Scripting Jython" icon="seti:python">
    Syntaxe Python 2.7 avec interop JVM complet. Compatible avec les scripts SikuliX existants sans modification.
  </Card>
  <Card title="Pipeline de vision" icon="puzzle">
    Primitives de vision par ordinateur modernes (template, feature, contour) composables depuis l'IDE ou par code.
